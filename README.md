# Object Detection and Tracking using YOLOv8

## 📋 Project Overview
This is **Task 4** of CodeAlpha AI Internship.  
A real-time object detection and tracking system built using **YOLOv8** (Ultralytics) and OpenCV. It can detect and track multiple objects from webcam or video files with tracking IDs.

## ✨ Features
- Real-time object detection using pre-trained YOLOv8 model
- Object tracking with persistent IDs
- Support for both webcam and uploaded video
- Bounding boxes with class labels and confidence scores
- Simple and clean Streamlit web interface

## 🛠️ Technologies Used
- Python
- Ultralytics YOLOv8
- OpenCV
- Streamlit (for user-friendly UI)
